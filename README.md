# unifyX Institutional Website 🚀

Este é o repositório oficial do site institucional do **unifyX**, a linguagem de programação universal do futuro.
Construído com extrema performance e um design system premium.

## 🛠️ Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Vite** (Build tool)
- **Tailwind CSS v4**
- **Lucide React** (Ícones)
- **Prism.js** (Syntax Highlighting)
- **React Router v7** (Roteamento)

## 📁 Estrutura do Projeto

\`\`\`text
src/
├── components/      # Componentes reutilizáveis (Button, Card, CodeBlock, etc.)
├── pages/           # Páginas principais (Home, Docs, Community, Blog, 404)

├── main.tsx         # Ponto de entrada do React

\`\`\`

## 🎨 Design System

O projeto utiliza um design system baseado no espaço de cores **OKLCH** para garantir brilho e constraste perfeitos:
- **Laranja Vibrante:** Primária, CTAs e destaques.
- **Pastel Peach:** Secundária e backgrounds de suporte.
- **Azul Escuro:** Tipografia base e elementos estruturais.

> A fonte _Poppins_ é utilizada para títulos, propiciando um tom amigável e tech, enquanto _Inter_ assume o corpo de texto e _JetBrains Mono_ os blocos de código.
